run - Выполнить команду
enter - Ввести текст в команду
type - Ввести символы в команду
ctrl - Ввести Ctrl+буква
alt - Ввести Alt+буква
keypad - Переключить клавиатуру для специальных клавиш
redraw - Принудительно обновить вывод
end - Отправить EOF (Ctrl-C) в команду
cancel - Аналогично /end
kill - Прервать процесс
status - Просмотр статуса и текущих настроее
cd - Изменить директорию
env - Управление переменными
shell - Изменить текущий терминал
resize - Изменить размер текущего терминала
setsilent - Включение/выключение вывода без звука
setlinkpreviews - Включение/выключение предпросмотра ссылок в выводе
setinteractive - Включение/выключение интерактивной оболочки
help - Получить справку
upload - Загрузить и перезаписать файл
r - Выполнить команду
token - Получить код доступа к боту
grant - Выдать доступ к боту
revoke - Забрать доступ к боту
